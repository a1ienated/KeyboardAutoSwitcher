# Keyboard Auto Switcher (Windows)

A lightweight Windows utility that lets you switch to your secondary keyboard layout with a hotkey and automatically revert to your default layout after a timeout.

[![Keyboard Auto Switcher tray icon](https://github.com/a1ienated/KeyboardAutoSwitcher/raw/main/KeyboardAutoSwitcher.png)](https://github.com/a1ienated/KeyboardAutoSwitcher/blob/main/KeyboardAutoSwitcher.png)

## Download
**Microsoft Store:** [Install](https://apps.microsoft.com/detail/9PCPX2N2R4LK)

## Key features
- Switch to a secondary layout via hotkey
- Auto-revert to your default layout after a timeout
- Minimal UI, runs in the tray
- Made for bilingual workflows (e.g., English/Russian)
- Works on Windows 10/11
